📖 Code du Travail de la RDC

Le Code du Travail de la République Démocratique du Congo constitue la base juridique des relations professionnelles entre employeurs et travailleurs.

Cette rubrique permettra la consultation des articles du Code du Travail, leur interprétation juridique et leur application pratique par les Inspecteurs et Contrôleurs du Travail.

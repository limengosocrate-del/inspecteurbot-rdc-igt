📖 Code du Travail RDC

📖 Présentation

Le Code du Travail de la République Démocratique du Congo constitue la base juridique des relations professionnelles entre employeurs et travailleurs.

Cette rubrique permet aux Inspecteurs et Contrôleurs du Travail d'accéder rapidement aux dispositions légales applicables.
